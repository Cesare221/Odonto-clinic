## Visao Geral

O Odonto-clinic foi desenvolvido como um projeto de portfolio voltado para o contexto de SaaS e software de gestao para clinicas. A proposta e centralizar informacoes importantes da operacao em uma unica interface, reduzindo retrabalho e melhorando a rotina de atendimento.

O sistema foi pensado para cobrir fluxos comuns de uma clinica odontologica, como organizacao de pacientes, agenda, acompanhamento de atendimentos e gestao interna.

## O que o sistema entrega

- cadastro e organizacao de pacientes;
- apoio a agenda e rotina de atendimentos;
- estrutura para gestao interna da clinica;
- interface web com foco em produtividade e clareza operacional.

## Stack identificada no repositorio

- JavaScript
- CSS

## Como rodar localmente

Revise os scripts e dependencias do projeto e execute a instalacao com o gerenciador de pacotes adotado no repositorio.

Exemplo comum:

```bash
npm install
npm run dev
```
